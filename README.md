# ARDE Interior Design Studio — Landing Page

This landing page was designed and developed to showcase the studio’s services, attract potential clients, and streamline the lead generation process.

It combines frontend development with simple backend-like automation to support real business needs.

## Key Features

- Responsive design (desktop-first approach)
- Clean and modern UI/UX
- Contact form for client inquiries
- Lead management workflow
- Integration with Telegram Bot API
- Data storage via Google Sheets (lightweight CRM)

## Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **Integration:** Telegram Bot API, Google Sheets API
- **Tools:** Axios, REST APIs

## Future Improvements

- Add analytics (user behavior tracking)
- Improve SEO optimization
- Extend CRM functionality
- Add multilingual support
